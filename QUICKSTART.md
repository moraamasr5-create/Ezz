## 🎯 في 3 دقائق فقط

### المرحلة 1️⃣: الفتح (30 ثانية)

```bash
# الطريقة 1: افتح الملف مباشرة
open index.html

# الطريقة 2: من متصفحك

# الطريقة 3: استخدم خادم محلي
python -m http.server 8000
# ثم زر http://localhost:8000
```

### المرحلة 2️⃣: الاستخدام الأول (2 دقيقة)

```
1️⃣ اختر الحلاق من القائمة المنسدلة
   👉 مثال: "أحمد محمد"

2️⃣ اضغط على الخدمات لإضافتها للسلة
   👉 مثال: قص شعر + حلاقة ذقن

3️⃣ راجع السلة على اليمين
   👉 ستجد السعر الإجمالي

4️⃣ اضغط "معاينة الفاتورة"
   👉 سترى كيف ستبدو الفاتورة المطبوعة

5️⃣ اضغط "إكمال البيع وطباعة"
   👉 ستُحفظ المعاملة وتُطبع الفاتورة

✅ تم! المعاملة محفوظة محلياً
```

## 📊 مراقبة الأداء

### مؤشرات صحية

```
✓ LocalStorage < 5MB    → كل شيء بخير
✓ Sync time < 5sec      → سريع جداً
✓ App load time < 2sec  → ممتاز
✓ Memory < 100MB        → طبيعي
```

### مؤشرات تحذير

```
⚠ LocalStorage > 5MB    → امسح البيانات القديمة
⚠ Sync time > 30sec     → تحقق من الإنترنت
⚠ App slow              → أعد تحميل الصفحة
⚠ Memory > 500MB        → أغلق المتصفح وافتحه
```
